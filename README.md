# nodejs-blog-website-complete
node js blog created with total.js, nosql, stylus, jquery and bootstrap

### Installation

Requires Node.js v4+, bower and stylus.

### Clone, install the dependencies and start the server.

```sh

$ npm install
$ bower install

--start dev

$ node debug.js 

--or for production

$ node release.js
```

Done
