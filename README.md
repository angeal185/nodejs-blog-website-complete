# nodejs-blog-website-complete
node js blog created with total.js, jquery, bootstrap and stylus 

### Installation

Requires Node.js v4+, bower and stylus.

Clone, install the dependencies and start the server.

```sh

$ npm install
$ bower install

###--start dev

$ gulp
$ node debug.js 

###--or for production

$ node release.js
```

Done
